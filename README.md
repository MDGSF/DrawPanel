# DrawPanel

## 设计文档

- 整个项目打包为一个独立的 React 组件，叫做 DrawPanel。
- 每个元素都设计为一个单独的 React 组件。
- 导入导出功能
  - 整个画布可以全部导出为一个 json 文件。
  - 可以把 json 文件再次导入。

### 元素类型

- 通用几何图形
  - 矩形 Rectangle
  - 正方形 Square
  - 圆形 Circle
  - 椭圆形 Ellipse
  - 三角形 Triangle
  - 菱形 Diamond
  - 平行四边形 Parallelogram
  - 小人 Actor
  - 线 Line
  - 曲线 Curve
- 矢量图形
- 贴图
- 文本
- ECharts
- AntDesign
